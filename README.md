# E-Commerce Project 🛒
Hi, I'm **Code Dreamer**, a 15-year-old developer. I believe that **simpler code is better than complex code**, and a **simple solution is better than a perfect one.**

This is a full-stack e-commerce application built with a focus on clean logic and essential functionality.

---

## 🛠 Tech Stack

* **Frontend:** Next.js, TailwindCSS, TypeScript
* **Backend:** Express, TypeScript, JWT (Auth)
* **Database:** PostgreSQL

---

## 📂 Project Structure

* `/e-commerce`: Frontend React/Next.js application.
* `/server`: Backend Express API.

---

## 🚀 Main Features & Pages

* **`/` (Shop):** Browse products, view details, and add to cart.
* **`/auth`:** Unified Login and Sign-in.
* **`/cart`:** Manage items (edit quantity, delete, or select/unselect for checkout).
* **`/checkout`:** A 3-step process (Shipping, Payment, Review).
* **`/order-history`:** View past orders.
* **`/admin`:** Admin dashboard 🚧 *(Under Development)*

---

## ⚠️ Current Status

* **Responsiveness:** Mobile responsiveness and accessibility features are not yet fully implemented.
* **Admin Page:** Currently under development — product management, order oversight, and user controls coming soon.

---

## ⚙️ Quick Start

1. **Backend:** `cd server && npm install && npm run dev`
2. **Frontend:** `cd e-commerce && npm install && npm run dev`
